自釀清酒套組

只要會煮飯，就能釀清酒！ （偷拿 憨啤酒 蘇董的slogon ）
自古以來，日本清酒的吟釀香一直都是釀造界的高嶺之花，香氣迷人卻又難以追求，而現在不一樣了！
這裡示範如何用最簡單的方式釀出清酒，期待有更多朋友一起加入，用台灣米，釀出台灣特色的美味！

包含：發酵罐 / 保溫紙箱 / 過濾袋 / 說明書 / 清酒材料包（米麴100g, 酒米200g, 清酒酵母）
應自備：消毒酒精 / 瓶裝水400ml / 煮飯工具 / 秤 / 1L有蓋廣口容器 / 12罐冷凍紙包飲料

本套組不含酒精 / 未滿18歲禁止飲酒 / 喝酒不開車開車不喝酒
